发表于2021-04-02
|阅读量:

Sass和Less介绍

  • Sass
  1. 兼容所有版本的 CSS
  2. 专业 CSS 扩展语言
  3. 使用变量
  4. 嵌套 CSS 规则
  5. 支持导入 SASS 文件
  6. 静默注释
  7. 混合器
  8. 使用选择器继承来精简 CSS
  • Less
  1. 增加了变量、Mixin、函数等特性
  2. 可以运行在 Node 或浏览器端

二、Sass和Less区别总结

  • Sass 语法类似 HTML，属于缩排语法，目的就是为了更快更好写，更容易阅读。Less 赋予了 CSS 动态语言的特性，如变量、继承、运算、函数，LESS 既可以在客户端上运行，也可以在服务端运行。

  • 变量符不一样，less 是 @，而 Sass 是 $

  • Sass 支持条件语句，可以使用 if{}else{}，for{} 循环等等。而 Less 不支持。
